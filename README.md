# Alledrogo
First version of my auction site project developed with Java 8, Spring and Angular. Made as a project for class at Poznan University of Technology. Now deprecated.


Technologies used:

- Spring 4
- Java 8
- Hibernate
- Spring ORM
- AngularJS
- Thymeleaf
- Quartz
- jQuery
- Spring Security
- Logback
- GSon
- MaterializeCSS
